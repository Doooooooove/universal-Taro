## 🚀 Quick Start

Follow these steps to get your Universal Tarot AI app running locally.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)

### Installation

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Configure Environment Variables**
   Set your API keys in the `.env.local` file:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

3. **Start the Development Server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000` to see your local application.
